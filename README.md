# 📌 Sugra Optical Website

## 🔗 Live Website  
https://sugraoptical.inrank.io/

---

## 📖 Project Description  
This project is a dynamic business website developed for an optical instruments company. It showcases various ophthalmic and optical products and provides an enquiry-based system instead of direct online purchasing.

---

## 🚀 Features  
- Product listing and detail pages  
- Custom enquiry cart system  
- Add to cart with product model selection  
- Contact/enquiry form with validation  
- CAPTCHA verification  
- Email-based enquiry system  
- Responsive design  

---

## 🛠️ Technologies Used  
- PHP (Core PHP)  
- MySQL  
- HTML, CSS, JavaScript  
- jQuery  

---

## 👨‍💻 My Role  
- Developed backend using PHP  
- Designed and managed database  
- Implemented enquiry cart using sessions  
- Added validations (email, phone, GST, CAPTCHA)  
- Integrated mail functionality  
- Worked on frontend integration  

---

## 🗄️ Database Setup  
1. Import `database.sql` into phpMyAdmin  
2. Configure database in `connection.php`  
3. Run project on localhost (XAMPP/WAMP)

---

## 📩 Key Functionality  
- Users can add products with models to enquiry cart  
- Submit enquiry with details  
- Enquiry is sent via email  

---

## ⚠️ Note  
- This is a live project currently hosted on a subdomain for demonstration and testing purposes. 
- Configure email using environment variable (`ADMIN_EMAIL`)
